# TimeStake

Time-weighted staking protocol with dynamic reward calculation on Base and Stacks blockchains.

## Features

- Stake tokens with time multiplier
- Dynamic weight calculation
- Time-based reward distribution
- Flexible unstaking
- Real-time reward calculation

## Tech Stack

- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Base**: Solidity 0.8.20, Foundry, Reown wallet
- **Stacks**: Clarity v4, Clarinet, @stacks/connect

## Getting Started

```bash
pnpm install
pnpm dev
```
